# **Robotics Concept Stack**

A robotics-oriented revision stack built from my master's degree coursework at Arizona State University.

---

Instead of treating each subject as an isolated academic module, this folder revisits them through a common robotics lens: modeling, control, perception, learning, autonomy, and system-level reasoning.

## Subjects

- **MAE 501: Linear Algebra in Engineering**  
- **MAE 506: Adv Sys Model, Dynamics & Ctrl**  
- **MAE 547: Modeling & Control of Robots**  
- **EEE 591: Reinforcement Learning Algorithms**  
- **CSE 598: Robot Perception (Computer Vision)**  
- **EEE 598: Generative-AI (Theory & Practice)**  
- **MAE 551: Appl ML for Mech & Aero Engineers**  
- **MAE 598: Multi-Robot System Dynamics**  
- **RAS 550: Mechatronic System Devices**  
- **SES 598: Space Robotics & AI**  

## What each summary contains

Each subject README is intended to capture:

- why the subject matters for robotics.
- the most important robotics-relevant concepts.
- mathematical or technical ideas worth retaining.
- links to real robotic systems, methods, or design problems.
- a compact revision-oriented takeaway.

## Why this exists

This is a personal attempt to distill the conceptual backbone of my master's degree into a structured and reusable robotics knowledge stack.
